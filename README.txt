To run this web app make sure to have at least php5 installed then, open a terminal and go inside the project root directory. Once you are here give:

		php -S localhost:8000

then, with your browser go to http://localhost:8000 to load the login page of the app. In the json provided, used in place of a database to minimize the environment setup for the tester, there are already some user saved with those emails:
